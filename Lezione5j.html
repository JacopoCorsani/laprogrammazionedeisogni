HTML

Di seguito è riportato un esempio di codice HTML con commenti:

Codice HTML Completo

<!DOCTYPE html> <!-- Specifica il tipo di documento come HTML5 -->
<html lang="it"> <!-- Inizio del documento HTML e lingua impostata su italiano -->
  <head> <!-- Inizio della sezione head per i metadati -->
    <meta charset="UTF-8"> <!-- Specifica la codifica dei caratteri -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- Garantisce un design responsivo -->
    <title>Lezione: Liste e Tabelle</title> <!-- Titolo della pagina mostrato nella scheda del browser -->
    <link rel="stylesheet" href="Esercizio.css"> <!-- Collega il file CSS per lo stile -->
  </head> <!-- Fine della sezione head -->
  <body> <!-- Inizio del corpo della pagina -->
    <header> <!-- Inizio dell'intestazione -->
      <h1>Ripasso delle Liste e Introduzione alle Tabelle</h1> <!-- Titolo principale della pagina -->
    </header> <!-- Fine dell'intestazione -->
    <main> <!-- Inizio della sezione principale -->
      <section> <!-- Sezione per le liste -->
        <h2>Ripasso: Liste</h2> <!-- Sottotitolo per le liste -->
        <h3>Lista non ordinata</h3> <!-- Sottotitolo per la lista non ordinata -->
        <ul> <!-- Inizio lista non ordinata -->
          <li>Elemento 1</li> <!-- Primo elemento della lista -->
          <li>Elemento 2 <!-- Secondo elemento della lista -->
            <ul> <!-- Lista annidata -->
              <li>Sottoelemento 2.1</li> <!-- Primo sottoelemento -->
              <li>Sottoelemento 2.2</li> <!-- Secondo sottoelemento -->
            </ul> <!-- Fine lista annidata -->
          </li>
        </ul> <!-- Fine lista non ordinata -->
        <h3>Lista ordinata</h3> <!-- Sottotitolo per la lista ordinata -->
        <ol> <!-- Inizio lista ordinata -->
          <li>Primo elemento</li> <!-- Primo elemento -->
          <li>Secondo elemento</li> <!-- Secondo elemento -->
          <li>Terzo elemento</li> <!-- Terzo elemento -->
        </ol> <!-- Fine lista ordinata -->
        <h3>Lista di definizione</h3> <!-- Sottotitolo per lista di definizione -->
        <dl> <!-- Inizio lista di definizione -->
          <dt>HTML</dt> <!-- Termine: HTML -->
          <dd>Linguaggio di markup per strutturare le pagine web.</dd> <!-- Definizione di HTML -->
          <dt>CSS</dt> <!-- Termine: CSS -->
          <dd>Linguaggio per definire lo stile delle pagine web.</dd> <!-- Definizione di CSS -->
        </dl> <!-- Fine lista di definizione -->
    </section> <!-- Fine sezione per le liste -->
    </main> <!-- Fine della sezione principale -->
  </body> <!-- Fine del corpo della pagina -->
</html> <!-- Fine del documento HTML -->

Stile CSS Utilizzato

Di seguito è riportato un esempio di codice CSS con commenti:

body {
  font-family: Arial, Helvetica, sans-serif; /* Imposta il font principale */
  margin: 0; /* Rimuove il margine predefinito del browser */
  background-color: #f9f9f9; /* Colore di sfondo chiaro */
  line-height: 1.6; /* Imposta l'altezza delle righe per una migliore leggibilità */
}

header {
  background-color: #4CAF50; /* Colore verde per l'intestazione */
  color: white; /* Colore del testo bianco */
  text-align: center; /* Centra il testo */
  padding: 20px; /* Aggiunge spazio interno */
}

h1, h2 {
  color: #333; /* Colore del testo scuro */
  text-align: center; /* Centra i titoli */
}

section {
  margin: 20px auto; /* Aggiunge margine sopra e sotto e centra la sezione */
  max-width: 800px; /* Imposta una larghezza massima */
  background-color: white; /* Colore di sfondo bianco */
  padding: 20px; /* Aggiunge spazio interno */
  box-shadow: 0px 2px 8px rgba(0, 0, 0, 0.1); /* Aggiunge un'ombra leggera */
}

ul, ol, dl {
  margin: 20px; /* Aggiunge margine attorno alle liste */
  padding: 10px; /* Aggiunge padding interno */
  background-color: #f4f4f4; /* Colore di sfondo chiaro */
  border: 1px solid #ddd; /* Bordo sottile grigio */
  border-radius: 5px; /* Arrotonda gli angoli */
  list-style-position: inside; /* Posiziona i marker delle liste all'interno */
}

ul ul, ol ol {
  margin-left: 20px; /* Indenta le liste annidate */
}

dl dt {
  font-weight: bold; /* Testo in grassetto per i termini della lista di definizione */
}

dl dd {
  margin-left: 20px; /* Indenta le descrizioni della lista di definizione */
}

table {
  width: 100%; /* Imposta la larghezza al 100% del contenitore */
  border-collapse: collapse; /* Rimuove gli spazi tra le celle */
  margin: 20px 0; /* Aggiunge margine sopra e sotto */
}

table, th, td {
  border: 1px solid #ddd; /* Bordo sottile grigio */
}

caption {
  font-weight: bold; /* Testo in grassetto per il titolo della tabella */
  margin-bottom: 10px; /* Aggiunge spazio sotto il titolo */
  text-align: center; /* Centra il titolo */
}

th {
  background-color: #4CAF50; /* Colore di sfondo verde per l'intestazione */
  color: white; /* Colore del testo bianco */
  padding: 10px; /* Aggiunge padding interno */
  text-align: left; /* Allinea il testo a sinistra */
}

td {
  padding: 10px; /* Aggiunge padding interno alle celle */
  text-align: left; /* Allinea il testo a sinistra */
}

tfoot td {
  font-weight: bold; /* Testo in grassetto per il footer */
  background-color: #f4f4f4; /* Colore di sfondo chiaro */
  text-align: right; /* Allinea il testo a destra */
}

footer {
  text-align: center; /* Centra il testo del footer */
  background-color: #4CAF50; /* Colore di sfondo verde */
  color: white; /* Colore del testo bianco */
  padding: 10px; /* Aggiunge padding interno */
  margin-top: 20px; /* Aggiunge margine sopra il footer */
}

Stile CSS Utilizzato

Di seguito è riportato un esempio di codice CSS con commenti:

table {
  border-collapse: collapse; /* Rimuove lo spazio tra le celle della tabella */
}

table, th, td {
  border: 1px solid; /* Aggiunge un bordo alle celle */
  border-color: black; /* Colore del bordo nero */
}

caption {
  font-weight: bold; /* Rende il testo del titolo della tabella in grassetto */
}

thead {
  color: green; /* Colore del testo per l'intestazione della tabella */
}

.menu {
  position: absolute; /* Posiziona il menu in un punto specifico della pagina */
  top: 10px; /* Distanza dal margine superiore */
  right: 10px; /* Distanza dal margine destro */
}

.menu ul {
  list-style: none; /* Rimuove i marker predefiniti delle liste */
  display: flex; /* Dispone le voci del menu in orizzontale */
  gap: 15px; /* Spaziatura tra le voci del menu */
}

.menu a {
  text-decoration: none; /* Rimuove la sottolineatura dai link */
  color: #333; /* Colore del testo dei link */
  font-weight: bold; /* Rende il testo in grassetto */
  transition: color 0.3s ease; /* Effetto di transizione per il colore al passaggio del mouse */
}

.menu a:hover {
  color: #007BFF; /* Cambia il colore dei link al passaggio del mouse */
}
