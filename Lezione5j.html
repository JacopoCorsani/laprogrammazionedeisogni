Esercizio sviluppato a lezione

HTML

Di seguito √® riportato il codice HTML sviluppato a:

<!DOCTYPE html>
<html>
  <head>
    <!-- Collegamento al file CSS per la stilizzazione della pagina -->
    <link rel="stylesheet" href="Esercizio.css" />
    <!-- Titolo principale della pagina -->
    <h1>Esercizio</h1>
    <!-- Titolo mostrato nella scheda del browser -->
    <title>Esercizio</title>
  </head>

  <body>
    <!-- Menu di navigazione -->
    <nav>
      <ul>
        <!-- Collegamenti alle diverse sezioni della pagina -->
        <li><a href="#sezione1">Vai alla sezione 1</a></li>
        <li><a href="#sezione2">Vai alla sezione 2</a></li>
        <li><a href="#sezione3">Vai alla sezione 3</a></li>
        <li><a href="#sezione4">Vai alla sezione 4</a></li>
        <!-- Collegamento a un'altra pagina -->
        <li><a href="page1.html">Vai alla pagina 1</a></li>
      </ul>
    </nav>

    <!-- Contenitore principale per le prime due sezioni -->
    <div class="section-container" id="menu">
      <!-- Prima sezione -->
      <div class="sezioni_affiancate" id="sezione1">
        <ul>
          <li>Prova</li>
          <li>Prova</li>
        </ul>
      </div>
    </div>
</body>
</html>
