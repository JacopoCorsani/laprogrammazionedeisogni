Sezione con Esempio di Codice usato a lezione

Stile CSS Utilizzato

Di seguito è riportato un esempio di codice CSS con commenti:

body, html {
  background-color: antiquewhite; /* Colore di sfondo della pagina */
  font-family: Arial, Helvetica, sans-serif; /* Font principale */
}

h1 {
  font-size: 64px; /* Dimensione del titolo principale */
  color: blue; /* Colore del testo */
  text-align: center; /* Testo centrato */
  font-family: 'Courier New', Courier, monospace; /* Font personalizzato */
}

img {
  border: 2px solid #ddd; /* Bordo grigio chiaro */
  border-radius: 8px; /* Arrotondamento degli angoli */
  width: 100%; /* Adatta la larghezza al contenitore */
  max-width: 250px; /* Limita la dimensione massima */
  height: auto; /* Mantiene le proporzioni */
  box-shadow: 2px 2px 8px black; /* Ombra nera */
  padding: 2px; /* Spaziatura interna */
}

a { /* Impostazioni grafiche per i link */
  color: red; /* Colore del testo rosso */
  text-decoration: none; /* Rimuove la sottolineatura */
}

a:hover { /* Settaggi quando si passa sul link con il mouse */
  color: black; /* Colore del testo nero quando si passa sopra */
  text-decoration: underline; /* Aggiunge una sottolineatura */
}
