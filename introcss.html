Cos'Ã¨ il CSS?

CSS (Cascading Style Sheets) Ã¨ un linguaggio utilizzato per descrivere l'aspetto e il layout di una pagina HTML. Permette di separare il contenuto (HTML) dallo stile (CSS).

Come collegare un file CSS

Ci sono tre modi per applicare il CSS:

1. Inline CSS

Stile applicato direttamente agli elementi HTML.

<p style="color: blue;">Testo blu</p>

2. Internal CSS

Stile definito all'interno di un tag <style> nella sezione <head>.

<style>
  p {
    color: blue;
  }
</style>

3. External CSS

File CSS separato collegato tramite il tag <link>.

<link rel="stylesheet" href="styles.css">

Consiglio: Utilizzare il metodo external CSS per mantenere il codice organizzato.
